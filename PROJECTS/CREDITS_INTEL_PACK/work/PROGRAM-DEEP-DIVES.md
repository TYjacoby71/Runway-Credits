## AWS ACTIVATE PROGRAM — FULL ANALYSIS

### Confirmed Facts (March 2026)
- **Max credits:** $100,000 (no tier variation mentioned)
- **Audience:** Startups at any stage (pre-seed through Series C+)
- **Key requirement:** No VC funding mandated (major advantage vs. competitors)
- **Partner ecosystem:** 140,000+ partners globally; 70% headquartered outside US
- **Core benefits:**
  1. AWS credits (offset infrastructure costs)
  2. Technical support
  3. Business mentorship
  4. Go-to-market co-selling via AWS Partner Network

### Application Requirements (Inferred 2026)
1. Valid business incorporation documents
2. Proof of startup status (cap table, angel funding docs, or accelerator acceptance letter)
3. Registered business email domain
4. LinkedIn company profile (signals legitimacy)
5. AWS Builder ID account (personal login)

### Timeline & Expiration
- **Approval time:** 1–2 weeks (fastest tier)
- **Credit validity:** 1–2 years from activation (creeping expiration risk)
- **Reapplication:** Can reapply after first batch expires (no stated limits)

### Known Limitations
1. **No affiliate rebate** — AWS doesn't pay referral fees for credit applications
2. **Usage restrictions:** Credits only apply to AWS services; not for 3rd-party software marketplace
3. **Cost control:** No automatic spending limits (must manage via IAM + budget alerts)
4. **Expiration cliff:** $100k → $0 on expiration date (no carryover)

### Strategic Value for Jacob's Stack
- **BatchTrack use case:** Database (RDS), compute (EC2/Lambda), storage (S3), potentially ML (SageMaker)
- **Tag & Ticket Sniper:** Lambda (heavy), API Gateway, CloudWatch
- **Recommendation:** Likely highest utilization rate; apply immediately

### Gotchas to Avoid
- Don't apply from a residential address (use business address)
- Don't apply under multiple entities simultaneously (AWS may reject as fraud)
- Keep AWS account separate from personal AWS usage (commingling risks)
